## 项目管理
- ``master`` 主分支；用户可下载使用
- ``alpha`` 开发数据分支(develop)，程序开发不直接操作``master`` 而由开发该分组再合并到主分支
- ``demo`` 项目实际测试；
- ``document`` 项目文档
- ``v{n}`` 历史版本分支，历史保存
